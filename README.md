# Talk Splunk Scripts

Talk Splunk 영상 채널에서 다루는 실습과 데모에 필요한 쉘 스크립트와 코드를 모아두는 저장소입니다. 각 스크립트는 영상과 동일한 환경을 재현하는 것을 목표로 하며, 실행 전에 내용을 확인하고 환경에 맞게 수정하는 것을 권장합니다.

## 구성
- `linux.sh`: Splunk Enterprise 9.4.7을 Linux 환경에 설치하고 부팅 시 자동 시작하도록 설정하는 설치 스크립트.
- `inst/`: 영상별로 추가 스크립트나 설정 파일을 저장할 예정인 디렉터리(빈 디렉터리).

## 사용 방법
- 실행 전 확인
  - 루트 권한이 필요합니다.
  - `wget`, `tar`가 설치되어 있어야 합니다.
  - 설치된 Splunk는 `/opt/splunk` 경로에 배치됩니다.
- 설치 실행 예시
  ```bash
  bash linux.sh
  ```
  설치 과정에서 라이선스 동의와 부팅 시 자동 시작이 자동으로 설정됩니다.

## 주의 사항
- 스크립트는 영상 촬영 시점의 버전과 경로를 기준으로 작성되었습니다. 운영 환경에 적용하기 전에 버전, 네트워크 경로, 권한 설정을 반드시 확인하세요.
- 자동 시작 설정이 불필요한 경우 `linux.sh` 마지막 줄을 수정하거나 제거하십시오.
